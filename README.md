# Sistema RAG Local con Node.js

## Tecnologías

- Node.js
- LanceDB
- Ollama
- Phi3
- Transformers.js

## Funcionalidades

- Ingesta multi PDF
- Chunking semántico
- Embeddings locales
- Retrieval semántico
- Base vectorial local
- Evaluación RAG
- Control de alucinaciones

## Pipeline

1. Lectura de PDFs
2. Limpieza de texto
3. Chunking
4. Embeddings
5. Persistencia vectorial
6. Retrieval
7. Generación con LLM

## Métricas

- Precision
- Recall
- Faithfulness
- Relevancia
